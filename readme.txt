This is the root folder.  We're using Eclipse 3.1.0 (www.eclipse.org).
The project tree is set up as follows:

docs/				documentation 
lib/				dependencies
metadata/			configuration files and other stuff
src/				source code
test/				testing (e.g. JUnit test) code
web/				web-related code
.classpath			classpath settings used by eclipse
.project			project settings used by eclipse
build.properties	default properties used by ANT
build.xml			ANT build file
properties.xml		properties used by ANT
readme.txt			this readme file
